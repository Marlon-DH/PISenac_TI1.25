## 📱 Aplicativos Android
### 🏛  Aplicativo Interativo para o Museu De Salto/SP

Projeto desenvolvido para o Museu de Salto, focado em acessibilidade e personalização da experiência dos visitantes no museu.

Aplicações desenvolvidas utilizando Android Studio e Kotlin, explorando conceitos de interfaces, navegação e persistência de dados.
